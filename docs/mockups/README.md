# AOT UI Mockups

ASCII mockups for the UI rewrite. Annotate with `<!-- @unc: your comment -->` inline
or add to the Comments section at the bottom of each file. Claude reads these and
updates the mockups.

## Files

- `01-run-list.md` — Default view, k9s-style run browser
- `02-new-run.md` — Input view: prompt → chat refinement → launch
- `03-run-detail.md` — Live agent activity feed + contextual tabs
- `04-components.md` — Shared component inventory (what to build vs buy)

## Conventions

- `@unc` — your annotations
- `@claude` — my responses to your annotations
- `[?]` — open question that needs your input
- `[!]` — known issue or concern
